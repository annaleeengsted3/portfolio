A mobile checkout flow

This student case was presented by magazine publishers Bonnier, and was an interesting case about further optimizing an already sleek mobile checkout flow and minimizing dropoff rate. Using Figma as a prototyping tool and user testing to compare ideas, we managed to remove an entire step from the flow and increase checkout speed. My role in this project was coding the working mobile prototype and design of the constant basket throughout the flow.

A redesign for a pattern designer

The goal of this student case was to redesign a designer's web solution with a strong, unified visual identity and identify and develop an online communication strategy. My role in particular was developing and coding the interactive and responsive web-prototype, utilizing WordPress for backend management, as well as implementing both technical and content SEO, and analysing the client's strategic and competetive positioning and customer relationships. Being the only pattern designer in the country using century-old techniques with a distinct graphic expression, we wanted to showcase our client's uniqeness and methods. Furthermore, part of unifying brand identity was to merge seperate portfolio and webshop domains into one single, uniform solution that communicated quality and exceptional pattern design.
